重要提醒：
不要在这里双击某个单独文件去运行或查看。
因为这会把单个文件临时解压到一个系统临时目录（普遍被称为 临时文件夹 或 缓存目录），关闭后会自动删除。
软件必须整体存在于一个固定文件夹中才能运行（普遍被称为 依赖关系）。
在压缩软件中执行“完整解压”操作
在压缩软件窗口上方寻找一个按钮，文字可能是：

解压过程中可能出现进度条或百分比显示。
等待到进度完成并窗口提示成功。
关闭压缩软件窗口后，前往你选择的解压位置（例如桌面或 D:\JackalClient）。
你应当看到一个新的文件夹，这个文件夹包含许多文件与子文件夹，这才是完整程序目录（普遍被称为 程序文件夹）。
处理杀毒软件误报并恢复被隔离文件
解压后，系统自带安全防护程序（普遍被称为 Windows Defender 或 杀毒软件）可能弹出提示，说发现威胁并已处理。
这些被处理的文件可能被移动到一个隔离存放的区域（普遍被称为 隔离区 或 恢复区）。
打开 Windows 安全中心：
点击任务栏的安全提示图标，或在开始菜单搜索“Windows 安全中心”。
进入“病毒和威胁防护”（普遍被称为 病毒扫描页面）。
找到“保护历史记录”或“威胁历史记录”（普遍被称为 历史记录列表）。

对每个条目执行“允许在设备上”“还原”等操作（普遍被称为 恢复）。
恢复时确认目标路径是你解压出来的固定文件夹，而不是压缩软件临时目录。
为解压后的整个文件夹添加信任（排除）
在 Windows 安全中心中找到“管理设置”。
找到“排除项”或“添加或删除排除项”（普遍被称为 白名单 或 信任列表）。
选择“添加排除项”，类型选择“文件夹”。
在弹出的文件夹选择窗口中，找到你解压的 JackalClient 文件夹。
鼠标左键单击该文件夹使其被选中，然后点击“选择文件夹”按钮。
这样系统会对该文件夹减少拦截，以防运行时被删文件。

在文件列表中寻找名为 “Launcher.exe” 的文件：
“.exe” 表示可执行程序文件（普遍被称为 可执行文件 或 程序）。

● その中から、ペンギンの絵、または「QQ」という文字が書かれているものを確認する。

● マウスを机の表面の上でゆっくり動かす。
● マウスを動かすと、画面上で小さな矢印や印が同じように動くことを確認する。
● この矢印状の表示は、一般に「マウスポインター」または「カーソル」と呼ばれる。
● マウスを動かし、カーソルの先端を「QQ」のアイコンの上に重ねる。
● マウスに付いている押し込み式の部品のうち、左側にあるもの（一般に「左ボタン」）を、人差し指で一度だけ押してすぐ離す。
● この操作は一般に「左クリック」と呼ばれる。
● もし画面に変化が起きない場合は、同じ左ボタンを短時間のうちに二回連続で押して離す。
● この操作は一般に「ダブルクリック」と呼ばれる。
● 操作が成功すると、四角い表示領域が画面上に現れる。
● この四角い表示領域は、一般に「ウィンドウ」と呼ばれ、ソフトウェアの内容が表示される。

手順2 目的のグループ（一般に「グループチャット」）に入る
● 表示された QQ のウィンドウの中を観察する。
● 左側または中央付近に、縦または横に並んだ項目の一覧があることを確認する。
● この一覧は、会話や参加している集団を表示するためのもので、一般に「チャット一覧」または「会話リスト」と呼ばれる。
● 一覧の中から、JackalClient に関するファイルが共有されているグループを探す。
● 一覧が長く、画面にすべて表示されていない場合がある。
● その場合、マウスの中央部分についている回転できる部品（一般に「スクロールホイール」）を指で回す。
● ホイールを回すと、一覧が上下に動くことを確認する。
● 目的のグループ名が画面内に表示されたら、カーソルをその文字の上に移動させる。
● マウス左ボタンを一度押して離し、そのグループを選択する。
● グループに入ると、多数の文字が表示された領域が見える。
● この領域は、過去の発言を表示する場所で、一般に「チャット履歴」と呼ばれる。
● その下または近くに、横に長い空白の入力用領域がある。
● これは文字を入力するための場所で、一般に「入力欄」または「入力ボックス」と呼ばれる。

手順3 グループ内のファイル一覧を開く
● グループチャットのウィンドウの上部または側面を見る。
● そこに複数の文字や項目が並んでいる帯状の部分がある。
● この部分は、一般に「機能バー」または「タブバー」と呼ばれる。
● その中から「ファイル」と書かれている項目、またはフォルダの形をした記号を探す。
● カーソルをその「ファイル」の文字または記号の上に移動させる。
● マウス左ボタンを一度押して離す。
● すると、グループ内で共有されているファイルの一覧が表示される。
● この一覧は、一般に「グループファイル一覧」と呼ばれる。

手順4 JackalClient という名前のフォルダを探して開く
● 表示されたファイル一覧の中を上から下まで順に確認する。
● 「JackalClient」という名前の項目を探す。
● その項目の横に、紙の束や箱のような絵が表示されていることを確認する。
● この絵が付いているものは、一般に「フォルダ」と呼ばれる。
● 一覧が長く画面に収まらない場合、再びスクロールホイールを使って上下に移動する。
● もしくは、画面右側にある細長い帯（一般に「スクロールバー」）を使う。
● スクロールバー上の小さな四角形（一般に「つまみ」）にカーソルを合わせる。
● マウス左ボタンを押したまま、上下に動かす。
● 目的の「JackalClient」フォルダが表示されたら、カーソルをその名前の上に移動させる。
● マウス左ボタンを短時間のうちに二回連続で押して離す。
● フォルダの中身が表示される。

手順5 最新バージョンの圧縮ファイルを確認する
● フォルダの中に複数のファイルが表示されていることを確認する。
● その中から、次のような名前のファイルを探す。
JackalClient_v0.8d_Pwd：Wormwake.zip
● 「JackalClient」はソフトウェアの名称である。
● 「v0.8d」はバージョン番号を示す文字列である。
● バージョン番号は、一般に数字や文字が新しいほど新しい内容を意味する。
● 「Pwd」はパスワードを意味する省略表記である。
● 「Wormwake」は解凍時に必要な文字列である。
● 「.zip」は圧縮されたファイル形式を示す拡張子である。
● カーソルをこのファイル名の上に移動させる。
● マウス左ボタンを一度押して離し、ファイルを選択状態にする。
● 選択されると、背景の色が変わることを確認する。

手順6 圧縮ファイルをダウンロードする
● 画面内に「ダウンロード」と書かれた操作項目があるか確認する。
● それが見つかる場合、カーソルを合わせて左クリックする。
● 見つからない場合、選択したファイルの上でマウス右ボタンを一度押す。
● すると、縦に並んだ選択肢の一覧が表示される。
● これは一般に「右クリックメニュー」または「コンテキストメニュー」と呼ばれる。
● その中から「ダウンロード」という文字を探す。
● マウス左ボタンでそれを選択する。
● ダウンロードが開始され、進行状況が表示される場合がある。
● ダウンロードが完了するまで待つ。
● 完了後、ファイルは通常、次の場所に保存される。
C:\Users\ユーザー名\Downloads
● この場所は一般に「ダウンロードフォルダ」と呼ばれる。
手順7 コンピュータ内のファイルを管理するための機能（一般に「ファイルエクスプローラー」と呼ばれるもの）を開く
● 画面の一番下の部分を確認する。
● 横に長く伸びた帯状の領域が表示されていることを確認する。
● この帯状の領域は、一般に「タスクバー」と呼ばれる。
● タスクバーの中に、黄色または白色のフォルダの形をした小さな絵を探す。
● この絵は、コンピュータ内のファイルを表示・管理するための機能を起動するためのものである。
● カーソルをそのフォルダ形の絵の上に移動させる。
● マウス左ボタンを一度押して離す。
● すると、新しい四角いウィンドウが画面に表示される。
● このウィンドウは、一般に「ファイルエクスプローラー」と呼ばれる。
● もしフォルダ形の絵が見つからない場合は、タスクバー左側にある Windows のロゴを探す。
● このロゴは、一般に「スタートボタン」と呼ばれる。
● カーソルをスタートボタンに合わせ、左クリックする。
● 表示された一覧の中から「ファイルエクスプローラー」という文字を探す。
● 見つけたら、それを左クリックする。

手順8 ダウンロードフォルダに移動する
● ファイルエクスプローラーのウィンドウの左側を見る。
● 縦に並んだ場所の一覧が表示されていることを確認する。
● この一覧は、一般に「ナビゲーションバー」または「サイドバー」と呼ばれる。
● その中から「ダウンロード」または「Downloads」という文字を探す。
● カーソルをその文字の上に移動させる。
● マウス左ボタンを一度押して離す。
● ウィンドウの右側の広い領域に、ダウンロードされたファイルの一覧が表示される。
● その中に「JackalClient_v0.8d_Pwd：Wormwake.zip」というファイルがあることを確認する。

手順9 Windows 標準の解凍機能を使用しないことを確認する
● この圧縮ファイルは、パスワード付きであることを思い出す。
● Windows に最初から備わっている解凍機能は、パスワード付きの圧縮ファイルを正しく処理できない場合がある。
● そのため、この方法では解凍を行わない。
● 必ず、圧縮専用のソフトウェア（一般に「圧縮ソフト」と呼ばれるもの）を使用する。
● 代表的な圧縮ソフトには、WinRAR、7-Zip、Bandizip などがある。
● 本手順では、これらのいずれかが既にインストールされていることを前提とする。

手順10 圧縮ソフトを使用して圧縮ファイルを開く（ただし中のファイルは直接開かない）
● 「JackalClient_v0.8d_Pwd：Wormwake.zip」のファイルの上にカーソルを移動させる。
● マウス右ボタンを一度押して離す。
● 画面に選択肢の一覧が表示される。
● この一覧は、一般に「右クリックメニュー」と呼ばれる。
● メニューの中から「WinRARで開く」「7-Zipで開く」「Bandizipで開く」などの項目を探す。
● 見つけた項目をマウス左ボタンで選択する。
● 圧縮ソフトのウィンドウが表示される。
● この状態は、圧縮ファイルの中身を“見るだけ”の状態である。
● この状態は、一般に「プレビュー表示」と考えてよい。
● この段階で、表示されている中のファイルを直接ダブルクリックしてはいけない。
● もし中のファイルを直接開くと、ファイルは一時的な場所にだけ展開される。
● この一時的な場所は、通常ユーザーが確認できない。
● ウィンドウを閉じると、その一時ファイルは自動的に削除される。
● ソフトウェアは、すべての関連ファイルが同じフォルダ内に存在しないと正常に動作しない。
● そのため、必ず「完全に解凍」する必要がある。

手順11 圧縮ファイルを完全に解凍する操作を行う
● 圧縮ソフトのウィンドウの上部を見る。
● 「解凍」「解凍先を指定」「Extract」「Extract To」などと書かれたボタンを探す。
● これらは、操作を実行するための押し込み式の部品である。
● カーソルをそのボタンの上に移動させる。
● マウス左ボタンを一度押して離す。
● 解凍設定用の小さなウィンドウが表示される。
● このウィンドウは、一般に「ダイアログ」と呼ばれる。

手順12 解凍先の場所を選択する
● ダイアログの中に、保存先の場所を示す欄があることを確認する。
● この欄には、現在選ばれているフォルダの場所が文字で表示されている。
● 解凍先は、後で自分が簡単に見つけられる場所を選ぶ。
● 例として、「デスクトップ」を選ぶ。
● ダイアログ内の一覧から「デスクトップ」という文字を探す。
● カーソルを合わせて左クリックする。
● 別の例として、D ドライブを選ぶこともできる。
● 「D:」と書かれた項目を選択し、その中に新しいフォルダを作成する。
● 新しいフォルダの名前は「JackalClient」と入力する。
● 解凍先が意図した場所になっていることを確認する。

手順13 解凍用パスワードを入力する
● 解凍を開始する際、パスワードの入力を求める画面が表示される。
● パスワード入力用の小さな四角い欄が表示される。
● カーソルをその欄の中に移動させる。
● マウス左ボタンを一度押して離す。
● 欄の中に縦線が点滅していることを確認する。
● これは、文字を入力できる状態であることを示す。
● キーボードを使って、次の文字列を正確に入力する。
Wormwake
● 最初の「W」は大文字で入力する。
● それ以外の文字は小文字で入力する。
● 余分な空白文字を入力しないよう注意する。
● 入力が終わったら、「OK」「解凍」「確定」などのボタンを探す。
● マウス左ボタンを一度押して離す。

手順14 解凍の完了を待ち、フォルダが作成されたことを確認する
● 解凍処理が始まると、進行状況を示す表示が現れる場合がある。
● 表示が完了するまで操作を行わずに待つ。
● 解凍が完了すると、エラーが出ずに処理が終了する。
● 圧縮ソフトのウィンドウを閉じる。
● 解凍先として指定した場所に移動する。
● 新しく作成された「JackalClient」というフォルダが存在することを確認する。
● そのフォルダの中に、多数のファイルとフォルダが入っていることを確認する。
● この状態が、ソフトウェアが正しく配置された状態である。

手順15 ウイルス対策ソフトによる誤検出への対応を行う
● 解凍後、画面右下などに警告表示が出る場合がある。
● これは、ウイルス対策ソフト（一般に「アンチウイルスソフト」）による通知である。
● Windows に標準で搭載されているものは「Windows セキュリティ」と呼ばれる。
● 通知をクリックするか、スタートメニューから Windows セキュリティを開く。
● 「ウイルスと脅威の防止」という項目を探す。
● その中に「保護の履歴」または「脅威の履歴」という項目がある。
● それを開く。
● JackalClient に関連するファイルが隔離されている場合がある。
● 各項目について、「許可」「復元」などの操作を行う。
● 復元先が、解凍した JackalClient フォルダであることを確認する。

手順16 JackalClient フォルダ全体を信頼対象として登録する
● Windows セキュリティの設定画面を開く。
● 「設定の管理」や「除外」などの項目を探す。
● 「除外を追加する」を選択する。
● 除外の種類として「フォルダ」を選ぶ。
● ファイル選択画面が表示される。
● 解凍した「JackalClient」フォルダを探す。
● そのフォルダを選択し、「フォルダの選択」をクリックする。
● これにより、フォルダ全体が信頼対象として登録される。
● 個別のファイルだけを登録するのでは不十分であることを理解する。

手順17 起動用プログラム（Launcher.exe）を実行する
● ファイルエクスプローラーで JackalClient フォルダを開く。
● 中に「Launcher.exe」という名前のファイルがあることを確認する。
● 「.exe」は、実行可能なプログラムであることを示す。
● カーソルを Launcher.exe の上に移動させる。
● マウス左ボタンを短時間のうちに二回連続で押して離す。
● 画面に確認の小さなウィンドウが表示される場合がある。
● これは、実行の許可を求める表示である。
● 「はい」または「Yes」を選択する。

手順18 別の起動方法：本体プログラムを直接実行する
● JackalClient フォルダの中に「Release」という名前のフォルダを探す。
● マウス左ボタンで二回連続して押し、フォルダを開く。
● 中に「JackalClient.exe」というファイルを探す。
● これがプログラム本体である。
● カーソルを合わせてダブルクリックする。
● 必要であれば、実行許可の確認で「はい」を選択する。

手順19 正常に起動したかを確認する
● プログラムが正常に起動すると、新しい画面や操作画面が表示される。
● 何も起きない場合は、次の点を確認する。
● ファイルがすべて解凍されているか。
● ウイルス対策ソフトによって削除されたファイルがないか。
● プレビュー状態の一時ファイルを実行していないか。
● 問題があれば、前の手順に戻って確認する。

手順20 最も重要な注意点の再確認
● 圧縮ファイルの中の個別ファイルを直接実行しない。
● 必ず「解凍」操作でフォルダ全体を展開する。
● 解凍パスワードは Wormwake である。
● ウイルス対策ソフトの隔離を確認し、復元と信頼設定を行う。
● 起動は Launcher.exe を優先する。
